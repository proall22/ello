# Ellol Digital Creatives

A premium, interactive, scroll-driven cinematic landing website designed for **Ellol Digital Creatives**. This project focuses on high-end visual storytelling, utilizing modern web technologies to create an immersive "Awwwards-level" experience.

## ✨ Key Features

- **Narrative-Driven Preloader**: An immersive intro sequence that sets the tone for the brand.
- **Cinematic Canvas Engine**: A high-performance GSAP-powered engine for playing back image sequences based on scroll progress.
- **Parallax Layer System**: A 3-depth parallax architecture (Background: 0.2x, Subject: 1x, Foreground: 1.4x) for enhanced spatial depth.
- **Minimalist Floating Navigation**: A clean, transparent header with a full-screen overlay menu and `mix-blend-difference` adaptive UI.
- **SEO Optimized**: Semantic HTML5 structure with proper heading hierarchy and ARIA labels for accessibility.
- **Interactive Services Grid**: Premium hover states with scale, translation, and custom glow shadows.
- **Functional Contact Engine**: Form validation with a Next.js API route for handling project inquiries.

## 🛠 Tech Stack

- **Framework**: [Next.js 14+](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/) (ScrollTrigger)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm / yarn / pnpm

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/ellocreatives.git
   cd ellocreatives
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
